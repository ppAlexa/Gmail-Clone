##  Overview
This project is a Gmail-inspired email inbox application that replicates core email functionalities. It was built to understand full-stack development by implementing a complete end-to-end system with a clean UI and scalable backend architecture.


##  Tech Stack

### Frontend
- HTML  
- Vanilla CSS  
- JavaScript  

### Backend
- Node.js  
- Express.js  
- RESTful APIs  

### Database
- MongoDB  


##  Architecture & Design

- RESTful APIs designed for all email operations  
- Modular controllers and routes for maintainable backend structure  
- MongoDB used for efficient data storage and retrieval  
- Clear separation between frontend and backend logic  


##  Core Functionalities

Create Mail : Compose and store new emails
Read Mail : Open emails and update read status
Star Mail : Toggle star state
Delete Mail : Remove emails from inbox
Inbox UI : Structured Gmail-like layout
